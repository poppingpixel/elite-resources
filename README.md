## 🛑 The Problem
Learning AI right now is a mess. You are drowning in scattered GitHub repos, unorganized ArXiv links, hundreds of Medium articles, and random Twitter threads. There is no clear path from fundamentals to cutting-edge research.

## 🌟 The Solution: Elite Resources
We built an **Apple-designed, high-fidelity React application** that maps out your entire learning journey. This isn't just a habit tracker; it's a comprehensive, interactive syllabus.

### ✨ What's Inside?
- **🧠 300+ Curated AI Topics**: A structured progression from Core Neural Networks and Transformers to advanced MLOps and AGI Safety.
- **📄 1,100+ Integrated Research Papers**: Instantly read summaries and access direct PDF links of the most seminal AI papers (AlexNet, Attention Is All You Need, Llama, etc.) right inside the topic modals.
- **🎯 CEO & Polymath Skills**: Expand beyond coding with executive leadership strategies and cross-disciplinary frameworks.
- **🏆 Progress Tracking**: Mark topics as "In Progress" or "Completed." Everything is saved locally in your browser—no login required.

## 🚀 Getting Started Locally

Want to run it yourself or contribute? It takes 30 seconds:

```bash
git clone https://github.com/poppingpixel/elite-resources.git
cd elite-resources
npm install
npm run dev
```
Navigate to `http://localhost:5173/` and start tracking your journey.

## 🤝 We Need Your Help (Contribute!)

We open-sourced this because the AI landscape moves too fast for one person to track. **We need the community to help us keep the roadmap elite.**

Check out our issues tab! We have plenty of **`good first issue`** tickets:
- Add new papers to `src/data/papers.ts`
- Fix typos in the topic descriptions
- Add new learning resources (YouTube links, courses) to `src/data/topicResources.ts`
